# **注册github**
## **登录github首页 <https://github.com/> 点击右上角的sign in**

## **依次输入自己的邮箱地址、注册密码、用户名，点击continue**


## **4次人机验证后，登录自己所填邮箱，输入GitHub给出的验证码。**

# **创建仓库**
## **点击create a repository**

## **输入项目信息：仓库名、选择公有或是私有、创建README文件**

# **上传文件**
## **点击红框中的upload files，选择自己的ppt进行上传。**
